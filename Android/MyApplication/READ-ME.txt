3)   Gradle requis/optionnelles
--   --------
Le gradle suivant doit être utilisé :
-Picasso

Picasso est un open source qui à était créer pour gérer les tailles des images
facilement
Télécharger : http://square.github.io/picasso/ 
Pour ceux qui utilise eclipse télécharger cette version "picasso-2.4.0.jar" dans 
le dossier lib, dans le cas ou vous utilisez Android Studio utilisez les dépencies
dans le build.gradle. 

dependencies {
	...
	compile "com.squareup.picasso:picasso:2.4.0"
	...
}

Il vous sera important de rajouter les permissions dans le fichier manifest si 
vos images proviennes d'internet. 
<uses-permission android:name"android.permission.INTERNET"/>
